#  Quiz App (C++)

A simple command-line Quiz Application built using 
Object-Oriented Programming in C++.

##  Features
- Multiple choice questions loaded from a file
- Score tracking
- Clean OOP structure (Question, Quiz, Player classes)

## Concepts Used
- Classes & Objects
- Encapsulation
- File Handling
- Arrays / Vectors

## How to Run
g++ main.cpp -o quiz
./quiz

##  Project Structure
quiz-app-cpp/
├── main.cpp
├── Question.h
